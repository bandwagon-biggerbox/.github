## 二、VPS 使用中常见的"安全风险"有哪些？

在正式介绍 BiggerBox 之前，先把问题拆清楚。用 VPS 跑业务，以下这几类问题是大多数用户会遇到的：

**线路不稳定导致的连接中断**
普通 CN2 GT 或者直接走公网的线路，在晚高峰时段经常出现延迟飙升、丢包严重的情况。数据同步、文件传输在这种环境下容易中途失败，需要反复重试。

**机房基础设施不可靠**
部分低价 VPS 商家的机房没有冗余电力、没有 RAID 存储，磁盘损坏即数据全失。这不是危言耸听，每年都有用户因此付出代价。

**IP 被封导致服务不可用**
对于国内访问的业务，IP 被封等于服务直接停摆。能不能快速换 IP、换了能不能退款，是评估一家 VPS 商的重要指标之一。

**没有备份机制**
数据无备份是最大的隐患。一旦误操作或者系统崩溃，数据找不回来。

**带宽不足引发的拥堵**
带宽不够，高峰期流量打满，上传下载速度骤降，业务可用性大打折扣。
